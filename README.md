💡 About the Project

VisionPrep is an AI-powered web platform designed to help visually impaired students prepare for exams independently, confidently, and without barriers.
Traditional learning platforms rely heavily on visual interaction — creating a huge gap for blind users.
VI exam protal  bridges this gap by providing a fully voice-enabled learning experience using cutting-edge AI technologies.

✨ With features like Text-to-Speech and Speech-to-Text, students can listen, speak, and learn — without ever needing to see the screen.

🚀 Features
🔊 Text-to-Speech (TTS)
Converts study material into natural-sounding audio
🎙️ Speech-to-Text (STT)
Allows users to answer questions and navigate using voice
🧠 AI Assistant
Helps solve doubts and guide users during learning
🧭 Voice Navigation
Fully hands-free platform — no mouse required!
📝 Accessible Mock Tests
Designed specifically for non-visual interaction
🌐 Inclusive UI/UX
Built with accessibility at the core (not as an afterthought)

🎯 Problem We Solve
📉 Millions of visually impaired students face challenges in:

Accessing study materials
Navigating online platforms
Attempting exams independently

👉 VisionPrep ensures equal learning opportunities by eliminating these barriers.

🛠️ Tech Stack
💻 Frontend: React.js / HTML / CSS
⚙️ Backend: Node.js / Express
🧠 AI Integration:
Text-to-Speech APIs
Speech Recognition APIs

🧪 How It Works
🎧 User logs in
🌐 Selects preferred language (Hindi or English)  
🔊 Content is converted into speech (TTS)
🗣️ User interacts using voice commands
✍️ Answers are captured via Speech-to-Text
🤖 AI assists in real-time for guidance
